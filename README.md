# 🐉 DragonBallApi

Este proyecto es una aplicación en Java que consume una API de Dragon Ball para obtener información sobre personajes, planetas y otros elementos del universo de Dragon Ball.

## Características

- 🔗 Consumo de una API RESTful de Dragon Ball.
- 📊 Procesamiento y visualización de datos obtenidos de la API.
- 🧱 Estructura modular utilizando Java y Gradle.

## 🛠️ Tecnologías utilizadas

- ☕ Java 17 o superior.
- ⚙️ Gradle como sistema de construcción.
- 📡 Librerías para consumo de APIs REST `Volley`

## 📋 Requisitos previos

- ✅ Java Development Kit (JDK) 17 o superior instalado.
- ✅ Gradle instalado o utilizar el wrapper incluido (`gradlew`).

## 🚀 Instalación y ejecución

1. Clona el repositorio: [🔗 GitHub](https://github.com/Keespp/DragonBallApi.git)
    
    ```bash
    git clone https://github.com/Keespp/DragonBallApi.git
    cd DragonBallApi
    ```
    
2. Construye el proyecto:
    
    ```bash
    ./gradlew build
    ```
    
3. Ejecuta la aplicación: [🔗 GitHub](https://github.com/Keespp/DragonBallApi.git)
    
    ```bash
    ./gradlew run
    ```

## 🧪 Uso

Una vez ejecutada la aplicación, esta realizará solicitudes a la API de Dragon Ball y mostrará la información obtenida en la consola o interfaz gráfica (dependiendo de la implementación). 🖥️📱
